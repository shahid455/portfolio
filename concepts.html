Concepts Learnt

Dynamic Memory Allocation

In the data structures and algorithm class, I learned about the importance of dynamic memory allocation in managing memory resources efficiently. Dynamic memory allocation allows us to allocate and deallocate memory at runtime, helping us work with data structures of varying sizes. We explored key concepts like malloc() and free() in C/C++, new and delete operators in C++, and memory management strategies to prevent memory leaks and fragmentation. Understanding dynamic memory allocation is crucial for creating flexible and efficient data structures and algorithms.

Linked Lists

One of the fundamental data structures I studied was the linked list. I gained a deep understanding of its types, including singly linked lists, doubly linked lists, and circular linked lists. The class covered basic insertion and deletion operations, and their advantages and limitations. Learning about linked lists was essential as they serve as a foundation for many other data structures and play a crucial role in developing algorithms for tasks like searching and sorting.

Stacks & Queue

Stacks are last-in, first-out (LIFO) data structures, and in my data structures and algorithms class, I comprehensively studied their applications and operations. We discussed how stacks are used in function calls, expression evaluation, and backtracking algorithms. Concepts like push and pop operations, stack overflow, and the implementation of stacks using arrays or linked lists were thoroughly covered. Queues, the first-in-first-out (FIFO) data structures, were another crucial topic in the course. I learned about the practical use of queues in scenarios like task scheduling, breadth-first search algorithms, and buffer management. The class explored basic queue operations like enqueue and dequeue, as well as circular queues and priority queues. The study of queues equipped me with the knowledge needed to design efficient solutions for problems involving ordered processing.

Trees

Trees are hierarchical data structures that I extensively studied in my data structures and algorithm class. I gained an understanding of binary trees, binary search trees, AVL trees, and B-trees, and their applications in various fields, including databases and file systems. The class covered tree traversal algorithms, balancing techniques, and the importance of maintaining ordered data. Learning about trees was instrumental in grasping advanced data structures and algorithms, as they often build upon tree-based structures.
